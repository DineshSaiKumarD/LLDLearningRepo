# Data Structures & Algorithms

This folder contains **DSA problem** solutions organized by topic.

## 📂 Folder Structure

```
DSA/
├── Arrays/              # Array-based problems
├── LinkedLists/         # Linked list problems
├── Trees/               # Binary trees, BST, etc.
├── Graphs/              # Graph algorithms
├── Sorting/             # Sorting algorithms
├── Searching/           # Search algorithms
├── Stacks/              # Stack-based problems
├── Queues/              # Queue-based problems
└── DynamicProgramming/  # DP problems
```

## 🚀 Getting Started

1. Navigate to a specific topic folder
2. Read the README in that folder
3. Review the solution code
4. Test with provided examples

## 📝 Problem Template

Each problem folder should contain:
- `Solution.java` - Implementation
- `README.md` - Problem description, approach, complexity analysis
- `TestCases.java` - Test cases
